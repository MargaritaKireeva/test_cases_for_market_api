# Тест-кейсы: Продукты (/products)

### 1. Успешное создание продукта

**Предусловие:** Сервер запущен. 

**Шаги:**
1. Отправить POST-запрос на `/products` с телом `{"name": <name>, "price": <price>}`.

**Ожидаемый результат:**
- Код ответа: `201 Created`
- В теле ответа и В БД `name` и `price` соответствуют переданным

**Наборы данных:**
- `name = "русское название"`, `price = 100.0`
- `name = "english name"`, `price = 100.0`
- `name = "name"`, `price = 1.0`
- `name = "a"`, `price = 100.0`
- `name = "name"`, `price = 0.01`
- `name = "name"`, `price = 99999999.99`

---

### 2. Ошибка 422 при создании продукта с пустым именем

**Предусловие:** Сервер запущен.

**Шаги:**
1. Отправить POST-запрос на `/products` с телом `{"name": <name>, "price": 100.0}`.

**Ожидаемый результат:**
- Код ответа: `422 Unprocessable Entity`

**Наборы данных:**
- `name = ""`
- `name = "  "`

---

### 3. Ошибка 422 при создании продукта с пустым телом запроса

**Предусловие:** Сервер запущен.

**Шаги:**
1. Отправить POST-запрос на `/products` с телом `{}`.

**Ожидаемый результат:**
- Код ответа: `422 Unprocessable Entity`
- В ответе две ошибки: для поля `name` и для поля `price`
- В каждой ошибке `msg` содержит `"Field required"`, `loc` указывает на соответствующее поле

---

### 4. Ошибка 422 при создании продукта с некорректным JSON

**Предусловие:** Сервер запущен.

**Шаги:**
1. Отправить POST-запрос на `/products` с телом `'{"name": "name", "price": }'`

**Ожидаемый результат:**
- Код ответа: `422 Unprocessable Entity`
- В `detail[0].msg` содержится текст `"JSON decode error"`

### 5. Ошибка 422 при создании продукта с невалидной ценой
**Предусловие:** Сервер запущен. 

**Шаги:**
1. Отправить POST-запрос на `/products` с телом `'{"name": "test", "price": <price>}'`

**Ожидаемый результат:**
- Код ответа: `422 Unprocessable Entity`

**Наборы данных:**
- `price = 0`
- `price = -0.01`
- `price = -1`

---

### 6. Успешное получение списка всех продуктов

**Предусловие:** Сервер запущен. Создан товар(`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**
1. Отправить GET-запрос на `/products`

**Ожидаемый результат:**
- Код ответа: `200 OK`
- Тип ответа: список
- Длина списка >= 1

---

### 7. Успешное получение продукта по id

**Предусловие:** Сервер запущен. Создан товар (`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**
1. Отправить GET-запрос на `/products/{<product_id>}`

**Ожидаемый результат:**
- Код ответа: `200 OK`
- `id`, `name`, `price` в теле ответа совпадают с переданным (`product_id`, `"name": "Test Product", "price": 100.0`)

---

### 8. Ошибка 404 при получении несуществующего продукта по id

**Предусловие:** Сервер запущен.

**Шаги:**
1. Отправить GET-запрос на `/products/99999`

**Ожидаемый результат:**
- Код ответа: `404 Not Found`
- В теле ответа: `{"detail": "Product not found"}`

---

### 9. Успешное удаление продукта

**Предусловие:** Сервер запущен. Создан товар (`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**
1. Отправить DELETE-запрос на `/products/{<product_id>}`
2. Отправить GET-запрос на `/products/{<product_id>}`

**Ожидаемый результат:**
- Код ответа DELETE-запроса: `200 OK`
- Код ответа GET-запроса: `404 Not Found`

---

### 10. Ошибка 404 при удалении несуществующего продукта

**Предусловие:** Сервер запущен.

**Шаги:**
1. Отправить DELETE-запрос на `/products/9999999999`

**Ожидаемый результат:**
- Код ответа: `404 Not Found`
- В теле ответа: `{"detail": "Product not found"}`

---

### 11. Успешное обновление одного или нескольких полей продукта

**Предусловие:** Сервер запущен. Создан товар (`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**
1. Отправить PATCH-запрос на `/products/{<product_id>}` с телом из набора данных.

**Ожидаемый результат:**
- Код ответа: `200 OK`
- В теле ответа измененные поля `name` и/или `price` соответствуют новым
- В теле ответа неизмененные поля `name` или `price` соответствуют изначальным

**Наборы данных:**
- `{"name": "Updated Name"}`
- `{"price": 250.0}`
- `{"name": "New", "price": 500.0}`

---

### 12. Ошибка 404 при обновлении несуществующего продукта

**Предусловие:** Сервер запущен. 

**Шаги:**
1. Отправить PATCH-запрос на `/products/99999` с телом `{"name": "test"}`.

**Ожидаемый результат:**
- Код ответа: `404 Not Found`
- В теле ответа: `{"detail": "Product not found"}`

---

### 13. Успешный запрос обновления продукта при пустом теле

**Предусловие:** Сервер запущен. Создан товар (`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**
1. Отправить PATCH-запрос на `/products/{<product_id>}` с телом `{}`.

**Ожидаемый результат:**
- Код ответа: `200 OK`

---

### 14. Ошибка 422 при обновлении цены продукта на невалидное значение

**Предусловие:** Сервер запущен. Создан товар (`product_id`, `"name": "Test Product", "price": 100.0`).

**Шаги:**

**Ожидаемый результат:**
- Код ответа: `422 Unprocessable Entity`

**Наборы данных:**
- `price = 0`
- `price = -0.01`
- `price = -1`
